# **Вариант №7. Сервис групп (Group Service)**

## Функции сервиса

### Добавить группу.

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| year | Год поступления | Обязательно | Целое число | от 2000 до 2999 | — |
| tutor_id | ID руководителя | Не обязательно | Целое число | строго больше 0 | `None` |
| student_count | Количество студентов | Не обязательно | Целое число | от 0 до 30 | 0 |
| cipher_of_the_training_area | Шифр направления подготовки | Обязательно | Строка | вида `NN.NN.NN` (цифры, разделённые точками) | — |
| number | Номер группы | Обязательно | Целое число | строго больше 0 | — |
| after_class_number | После какого класса поступили | Обязательно | Целое число | строго 9 или 11 | — |
| prefix | Префикс | Обязательно | Строка | длина строки от 1 до 2 | — |

#### Информация, возвращаемая в случае удачного запроса

| Параметр | Пояснение | Тип |
|---|---|---|
| id | ID группы | Целое число |
| year | Год поступления | Целое число |
| is_active | Статус активности группы | Булевый тип |
| tutor_id | ID руководителя | Целое число |
| student_count | Количество студентов | Целое число |
| cipher_of_the_training_area | Шифр направления подготовки | Строка |
| number | Номер группы | Целое число |
| after_class_number | После какого класса поступили | Целое число |
| prefix | Префикс | Строка |

### Удаление группы по ID

| Параметр | Пояснение | Обязательность | Тип | Ограничение | 
|---|---|---|---|---|
| id | ID группы | Обязательно | Целое число | > 0 |

> Фактически запись из БД не удаляется, а помечается как неактивная (через булевое поле `is_active`).

#### Информация, возвращаемая в случае удачного запроса

> {"deleted": True}

### Получить группы по заданным параметрам

| Параметр | Пояснение | Тип | 
|---|---|---|
| year | Год поступления, фильтрация (меньше указанного, равно указанному, больше указанного) | Целое число |
| tutor_id | ID руководителя, фильтрация (равно указанному) | Целое число | 
| is_active | Статус активности группы, фильтрация (`True` или `False`) | Булевый тип |
| student_count | Количество студентов, фильтрация (меньше указанного, равно указанному, больше указанного) | Целое число |
| cipher_of_the_training_area | Шифр направления подготовки, фильтрация (равно указанному) | Строка |
| number | Номер группы, фильтрация (равно указанному) | Целое число | 
| after_class_number | После какого класса поступили, фильтрация (равно указанному (9 или 11)) | Целое число |
