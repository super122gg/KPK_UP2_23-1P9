# Документация API Discipline Service

## Сущность: `Discipline`

---

## 1. Добавить Discipline

### Информация для создания Discipline

| Параметр (на английском) | Пояснение                                               | Обязательность | Тип     | Ограничение                                      | Значение по умолчанию |
| ------------------------ | ------------------------------------------------------- | -------------- | ------- | ------------------------------------------------ | --------------------- |
| `name`                   | Название дисциплины                                     | Да             | string  | max_length=100                                   | -                     |
| `code`                   | Код дисциплины                                          | Да             | string  | max_length=20                                    | -                     |
| `total_hours`            | Общее количество часов на дисциплину                    | Да             | integer | > 0                                              | -                     |
| `category_id`            | Идентификатор категории, к которой относится дисциплина | Да             | integer | Должен существовать в таблице Category, not null | -                     |

### Уникальные комбинации параметров

- `name` — название дисциплины должно быть уникальным
- `code` — код дисциплины должен быть уникальным

### Информация, возвращаемая при успешном создании Discipline

| Параметр (на английском) | Тип     |
| ------------------------ | ------- |
| `id`                     | integer |
| `name`                   | string  |
| `code`                   | string  |
| `total_hours`            | integer |
| `category_id`            | integer |

---

## 2. Изменить Discipline по ID

### Информация для изменения Discipline по ID

| Параметр (на английском) | Пояснение                     | Обязательность | Тип     | Ограничение                    | Значение по умолчанию |
| ------------------------ | ----------------------------- | -------------- | ------- | ------------------------------ | --------------------- |
| `name`                   | Новое название дисциплины     | Нет            | string  | max_length=100, уникальное     | -                     |
| `code`                   | Новый код дисциплины          | Нет            | string  | max_length=20, уникальный      | -                     |
| `total_hours`            | Новое количество часов        | Нет            | integer | > 0                            | -                     |
| `category_id`            | Новый идентификатор категории | Нет            | integer | Должен существовать в Category | -                     |

> **Примечание:** Хотя бы один параметр должен быть передан для изменения.

### Информация, возвращаемая при успешном изменении Discipline

| Параметр (на английском) | Тип     |
| ------------------------ | ------- |
| `id`                     | integer |
| `name`                   | string  |
| `code`                   | string  |
| `total_hours`            | integer |
| `category_id`            | integer |

---

## 3. Удалить Discipline по ID

**Возвращаемое значение:** `True`, если дисциплина была удалена (закрыта), иначе `False`

| Сценарий                                       | Возвращаемое значение |
| ---------------------------------------------- | --------------------- |
| Дисциплина с указанным ID существует и удалена | `True`                |
| Дисциплина с указанным ID не найдена           | `False`               |

---

## 4. Получить Discipline по ID

### Информация, возвращаемая при успешном поиске Discipline по ID

| Параметр (на английском) | Пояснение                           | Тип     |
| ------------------------ | ----------------------------------- | ------- |
| `id`                     | Уникальный идентификатор дисциплины | integer |
| `name`                   | Название дисциплины                 | string  |
| `code`                   | Код дисциплины                      | string  |
| `total_hours`            | Общее количество часов              | integer |
| `category_id`            | Идентификатор категории             | integer |

---

## 5. Получить список Discipline по заданным параметрам

### Информация для получения списка Discipline

| Параметр (на английском) | Пояснение                      | Тип     | Описание                                                         |
| ------------------------ | ------------------------------ | ------- | ---------------------------------------------------------------- |
| `category_id`            | Фильтрация по категории        | integer | Необязательный. Возвращает дисциплины только указанной категории |
| `name_contains`          | Поиск по части названия        | string  | Необязательный. Регистронезависимый поиск                        |
| `code_contains`          | Поиск по части кода            | string  | Необязательный                                                   |
| `min_hours`              | Минимальное количество часов   | integer | Необязательный. Фильтр >= указанного значения                    |
| `max_hours`              | Максимальное количество часов  | integer | Необязательный. Фильтр <= указанного значения                    |
| `limit`                  | Ограничение количества записей | integer | Необязательный. По умолчанию = 100                               |
| `offset`                 | Смещение для пагинации         | integer | Необязательный. По умолчанию = 0                                 |

### Информация, возвращаемая в виде списка Discipline

| Параметр (на английском) | Тип     |
| ------------------------ | ------- |
| `id`                     | integer |
| `name`                   | string  |
| `code`                   | string  |
| `total_hours`            | integer |
| `category_id`            | integer |

> Возвращается **список** объектов с указанными выше полями.
